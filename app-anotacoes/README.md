**Esse projeto foi desenvolvido para colocar em prática o conhecimento adquirido no React Native com os seguintes recursos: UseState, UseEffect e AsyncStorage.**

### Recursos externos (não nativos) aplicados

- `SafeAreaView` from react-native-safe-area-context
- `AsyncStorage` from react-native-async-storage/async-storage

### Como instalar os recursos externos e nativos?

Com a pasta do projeto aberta em seu terminal, insira o seguinte comando: `npm i` (esse comando vai instalar todas as depêndencias necessárias para que o projeto funcione).

## A respeito de cada recurso utilizado

**useState**
Onde você guarda informações que podem mudar ao longo do tempo, como o número de itens em um carrinho de compras ou o texto digitado em um campo de entrada..

**useEffect**
Ele permite que você execute algumas tarefas sempre que seu componente muda ou quando algo específico acontece.

**AsyncStorage**
É uma ferramenta que você usa para armazenar dados de forma persistente no dispositivo do usuário
